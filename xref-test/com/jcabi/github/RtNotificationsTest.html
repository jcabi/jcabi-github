28       * Method 'iterate()' returns empty iterable if the service responds with

31      @Test
32      void iterateEmpty() {

34              "Collection is not empty",
35              new RtNotifications(
36                  new FakeRequest()
37                      .withBody("[]")
38              ).iterate(),
39              Matchers.emptyIterable()
